# 自动文本翻译器

基于 DeepSeek 大语言模型的 CSV 术语批量翻译工具，专为电气自动化领域的专业术语翻译设计，支持动态多语言列扩展。

## 功能特性

- **双击即用**：双击 `TextTranslate.py` 即可运行，无需命令行操作
- **批量翻译**：一次性将所有待翻译条目发送给 DeepSeek API，高效完成翻译
- **多语言支持**：自动识别表头中所有含 `【翻译】` 标记的列，支持同时翻译为英文、日语、世界语等多种语言
- **增量跳过**：已有翻译内容的单元格自动跳过，支持重复运行
- **弹窗提示**：翻译完成或出错时，通过 Windows 原生弹窗 + 命令行双重提示
- **密钥本地存储**：API Key 保存在本地 `API_key.key` 文件中，首次运行时会引导输入

## CSV 文件格式

```csv
序号,中文文本,英文【翻译】,日语【翻译】,世界语【翻译】
1,启动按钮,Start button,起動ボタン,Startbutono
2,停止按钮,Stop button,,
3,急停按钮,Emergency stop button,,
```

- **第 1 列**：序号
- **中文文本列**：列名需包含"中文"和"文本"，作为翻译源
- **目标语言列**：列名格式为 `{语言名}【翻译】`，例如 `英文【翻译】`、`日语【翻译】`、`世界语【翻译】`
- 目标列中已有内容的单元格会被自动跳过，空白单元格才会翻译
- 可随时手动在表头添加新的语言列，脚本会自动识别

## 快速开始

### 1. 环境要求

- Python 3.7+
- Windows 操作系统

### 2. 安装依赖

```bash
pip install openai
```

### 3. 运行

双击 `TextTranslate.py` 即可运行。

首次运行时会弹出对话框，提示输入 DeepSeek API Key，输入后会自动保存到 `API_key.key` 文件中，后续运行不再需要重新输入。

### 4. 获取 API Key

前往 [DeepSeek 开放平台](https://platform.deepseek.com/) 注册并获取 API Key。

## 注意事项

- 翻译结果依赖大模型输出，建议人工校对专业术语的准确性
- `API_key.key` 包含敏感信息，建议将其添加到 `.gitignore`
- 翻译提示词针对电气自动化领域优化，其他领域的术语可能需要调整 system prompt
